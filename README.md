# Md5-decrypt

 Decryption md5 hash on Cuda. 
 Based on the md5 hash and the length of its word, it returns the word contained in the hash.
 School exercise.

 ## Usage

 * In terminal goes to directory wtih project:

 ```bash
   make
   ```

* Run:

```bash
   ./main1 [number of words in hash] [hash] 
   ```
